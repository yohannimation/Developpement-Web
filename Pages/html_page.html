HTML

Dans cette page, vous découvrirez tout ce que j'ai pu apprendre en HTML.

Les bases


  • h1 /h1, h2 /h2, ..., h6 /h6 : créer des titres. h1 étant le plus gros et h6 le plus petit
  • p /p : permet de créer un paragraphe
  • hr : insert une ligne de séparation
  • br : fait un retour à la ligne
  • pre /pre : affiche les lignes de code tel quels sont dans le code
  • ul /ul : (unordered list) liste sans ordre particulier
  • ol /ol : (ordered list) liste ordonnée
  • li /li : (list item) élément de liste
  • a /a : pour faire une redirection avec un lien
  • form /form : permet de créer un formulaire
  • table /table : créer un tableau

Les médias


  • img : insert une image dans la page
  • figure /figure : regroupe une ou plusieurs image(s)
  • figcaption /figcaption : ajoute une légende
  • audio /audio : créer un élément audio
  • video /video : permet de mettre une vidéo dans la page
  • source / : plusieurs balise peuvent êter inséré sous un élément audio ou vidéo pour avoir plusieurs formats de médias en secours
  • iframe /iframe : permet d'intégrer un document externe (vidéo YouTube, carte Google Map, ...)

La sémantique


Les balises sémantiques permettent de donner un sens à la page web. En partie pour les utilisateurs ayant des aides pour naviguer sur une page web, mais aussi pour les intelligence artificiel du web qui "regarde" si ces balises sont correctement référencé. Si elles le sont, le site internet sera mieux mis en avant.

  • main /main : contenu principal
  • section /section : zone de la page traitant d'un même thème
  • article /article : contenu pouvant se suffire à lui même
  • header /header : en-tête (de la page, d'un article, d'une section, ...)
  • footer /footer : pied (de la page, d'un article, d'une section, ...)
  • thead /thead : en-tête pour les tableaux
  • tfoot /tfoot : pied pour les tableaux
  • tbody /tbody : corps du tableau
  • nav /nav : élément de navigation (menu, fil d'ariane, index, …)

Je ne vais pas montrer d'exemple d'utilisation pour chaque balise, vous pourrez découvrir leur utilisation dans les pages suivantes.
